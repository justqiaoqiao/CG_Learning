# 作业二:旋转与变化

一个简单、可交互的 3D 变换演示程序，基于 **Taichi** 实现，直观展示 MVP（模型-视图-投影）变换流程。  
你可以通过键盘控制三角形绕 Z 轴旋转，观察 3D 坐标如何映射到 2D 屏幕上。

## 作业：三角形的变换
### 三维三角形顶点与任务描述

在提供的代码框架中，我们定义了一个三维空间中的三角形。该三角形的初始顶点坐标为：

- v0=(2.0,0.0,-2.0)
- v1=(0.0,2.0,-2.0)
- v2=(-2.0,0.0,-2.0)

通过 **MVP 矩阵** 变换为二维屏幕坐标，并在 Taichi 的 GUI 窗口中绘制出对应的线框三角形。

### 目标
1. 深入理解 3D 空间中的坐标变换流程（模型-视图-投影/MVP(Model-View-Projection)变换）。
2. 独立推导并用代码实现模型变换（Model）、视图变换（View）和投影变换（Projection）矩阵。
3. 掌握面向数据编程框架 Taichi 的基本语法与矩阵操作。
